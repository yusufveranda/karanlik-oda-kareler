# karanlik-oda-kareler

[Karanlık Oda](https://verandatools.com/karanlik-oda/) günlük film tahmin oyununun sahne ipucu kareleri.

- Her klasör bir TMDB film id'si; içinde `1.webp` → `5.webp` (az ele verici → ikonik sırayla).
- Kaynak: resmî tanıtım materyalleri (fragman kareleri ve basın/galeri görselleri). Telif ilgili stüdyolara aittir; eğitim/eğlence amaçlı, ticari olmayan kullanım.
